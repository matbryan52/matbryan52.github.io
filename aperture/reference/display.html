class aperture.display.vectors.VectorsOverlay(*args, fig: BokehFigure = None, fill_values: dict[str, Any] | None = None, name: str | None = None, **kwargs)[source]

Bases: GlyphBase

Adds two lines to the figure at right-angles, with options to control color and add text labels. Also has functionality to interactively set the vector angle using a slider.

If initialized on a PointSet, vector origin location will update based on either changes to the PointSet data source, or to the value of the selection box returned by point_select().

This class currently requires the user to call setup_callbacks() to configure which types of interactivity should be created. This should be the subject of a future refactoring as needed.

Parameters

  • pointset (PointSet, optional) – A PointSet to use as the source for the vector origin, by default None in which case pos_xy must be supplied.

  • pos_xy (tuple[float, float], optional) – Values to use for the vector origin, by default None in which case pointset must be supplied.

  • length (float, optional) – The vector length in data units, by default 100.

  • colors (tuple[str, str], optional) – A 2-tuple of color strings, one for each vector, by default None in which case colors are red and blue.

  • labels (tuple[str, str], optional) – 2-tuple of label strings to applied to the end of each vector, by default None in which case there are no labels applied.

  • text_length_mult (float, optional) – A multiplier applied to length to defined text label origin, by default 1.1

  • initial_idx (int, optional) – The initial index in the pointset datasource to which to attach the vector origin, by default 0

  • initial_angle (float, optional) – The inital angle of the vectors in radians, relative to a horizontal-right origin, by default 0.

  • labels_kwargs (dict, optional) – Optional kwargs passed to the TextOverlay constructor, which can be used to set style options on the labels

  • **kwargs (dict, optional) – Optional kwargs passed to the Bokeh MultiLine constructor, which can be used to set style options on the vector glyph.
