Wrapper for a bokeh.plotting.figure object

  • Handles initialization of plots, glpyhs etc

  • Handles figure-level elements (axis labels, size/shape)

This is a base class which doesn’t strictly rely on Panel, it could be used in a pure Bokeh environment if you insert self.fig into the current Bokeh document. The child BokehFigure which inherits this base class is a full Panel component.

class libertem_ui.display.figure.BokehFigure(**kwargs)[source]

Bases: libertem_ui.display.figure.BokehFigureBase, panel.pane.plot.Bokeh

BokehFigureBase which is a standalone Panel component

This class subclasses pn.pane.Bokeh which means it is treated like a standard Panel component able to be inserted into layouts and also displayed directly in a notebook or made .servable()

As a result it is also a param.Parametrized class with default Parameters which come from Panel. Currently these parameters are not taken into account or set by the BokehFigure class, but this could be harmonized in the future with a bit of refactoring.
