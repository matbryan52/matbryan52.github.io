class aperture.udf.imaging.Imaging_UI[source]

Bases: Imaging_UIBase

Create a UI panel to perform virtual imaging on a LiberTEM dataset

The user can interactively browse the diffraction patterns (a Pick UDF is automatically run each time the user interacts), configure the imaging (mask type, radii, centre etc.) then launch the analysis with iterative updating and a progress bar. The analysis runs asynchronously so the UI should remain responsive during computation.

Supports point-, disk- and ring- LiberTEM mask analyses, and automatically adds functionality to draw one-or-more rectangle ROIs on the navigation pane

classmethod panel(ctx: lt.Context, ds: lt.DataSet, results_container: ResultTree | None = None, page_title: bool | str = True) → AutoTwoPane[source]

Build the UI for the specific dataset and setup the analyses using the suplied context

The returned layout object can be served if running with panel serve, interactively launched using .show() or displayed directly in a notebook by returning it from a cell

Parameters

  • ctx (lt.Context) – LiberTEM context object

  • ds (lt.DataSet) – LiberTEM dataset object

  • results_container (Optional[ResultTree]) – A ResultTree object which the UI will fill with results and metadata from the analysis run

  • page_title (Union[bool, str]) – If a string, set this as the layout title, else use a default title if True. Set no page title if False.
