A UI toolkit based on the Panel and Bokeh libraries, focused on image data processing.

Intended to support the LiberTEM library for electron microscopy data analysis.

Get started

In a Jupyter cell, create a figure containing a 2D numpy.array as an image:

import aperture as ap
ap.image(np_img)

Create a figure with interactive polygon drawing tools:

fig = ap.figure()
polygons = fig.add_polygons(allow_edit=True)
fig  # to trigger display at the end of the cell

Export the polygons as pandas.Dataframe:

polygons.export_data()

Features

  • Display toolkit for rich, interactive figures

  • Figure configuration tools automatically provided

  • Image display with auto-downsampling (datashading)

  • Drawing tools (points, boxes, polygons etc) with simple export to pandas DataFrame

  • Optimizing layout engine for groups of tall/wide/square figures with accompanying widgets

  • Workflow interface chaining multiple GUI panels to create interactive pipelines

  • Results management with auto-generation of save/export widgets

Design goals

  • Single page webapp format or Jupyter notebook

  • Equivalent UI panels in a Jupyter Notebook with minimal adapter code

  • Easy to save and export results in a useful format

  • Modular, easy to add new components

  • No Javascript/Typescript required unless desired for performance reasons

Installation

With pip directly from GitHub

pip install https://github.com/matbryan52/aperture.git
